# 🧩 Cards de História – TripWise

---

## 🟩 US01 – Criar viagem

**História:**

> Como usuário, quero cadastrar uma viagem informando destinos e datas, para planejar meu roteiro.

**Critérios de Aceitação:**

* Deve permitir inserir um ou mais destinos
* Deve permitir definir datas de início e fim
* Deve salvar a viagem no sistema
* Deve validar campos obrigatórios

---

## 🟩 US02 – Definir preferências

**História:**

> Como usuário, quero definir minhas preferências (cultura, gastronomia, lazer), para receber sugestões personalizadas.

**Critérios de Aceitação:**

* Deve permitir selecionar múltiplas preferências
* Deve salvar as preferências do usuário
* Deve utilizar as preferências na geração do roteiro

---

## 🟩 US03 – Gerar roteiro

**História:**

> Como usuário, quero gerar automaticamente um roteiro de viagem, para economizar tempo no planejamento.

**Critérios de Aceitação:**

* Deve gerar o roteiro automaticamente
* Deve considerar destinos e preferências
* Deve organizar atividades por dia
* Deve exibir o roteiro ao usuário

---

## 🟩 US04 – Visualizar custos

**História:**

> Como usuário, quero visualizar os custos estimados da viagem, para controlar meu orçamento.

**Critérios de Aceitação:**

* Deve exibir custo de hospedagem
* Deve exibir custo de transporte
* Deve exibir custo de atrações
* Deve apresentar um custo total estimado

---

## 🟩 US05 – Comparar transportes

**História:**

> Como usuário, quero comparar opções de transporte, para escolher a melhor alternativa entre custo e tempo.

**Critérios de Aceitação:**

* Deve listar diferentes opções de transporte
* Deve exibir tempo de deslocamento
* Deve exibir custo estimado
* Deve permitir comparação entre opções
