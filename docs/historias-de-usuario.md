📌 🇧🇷 Histórias de Usuário
🟢 US01

Como usuário, quero cadastrar uma viagem informando destinos e datas, para planejar meu roteiro.

🟢 US02

Como usuário, quero definir minhas preferências (cultura, gastronomia, lazer), para receber sugestões personalizadas.

🟢 US03

Como usuário, quero gerar automaticamente um roteiro de viagem, para economizar tempo no planejamento.

🟢 US04

Como usuário, quero visualizar os custos estimados da viagem, para controlar meu orçamento.

🟢 US05

Como usuário, quero comparar opções de transporte, para escolher a melhor alternativa entre custo e tempo.



📌🇺🇸 User Stories 
🟢 US01

As a user, I want to create a trip with destinations and dates, so I can plan my itinerary.

🟢 US02

As a user, I want to set my preferences, so I can receive personalized suggestions.

🟢 US03

As a user, I want the system to generate a travel itinerary automatically, so I can save time.

🟢 US04

As a user, I want to view estimated travel costs, so I can manage my budget.

🟢 US05

As a user, I want to compare transport options, so I can choose the best option based on cost and time.
