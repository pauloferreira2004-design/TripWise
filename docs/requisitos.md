# 📌 Requisitos do Sistema – TripWise

---

## 🇧🇷 Requisitos Funcionais

**RF01 – Cadastro de viagem**  
O sistema deve permitir que o usuário cadastre uma nova viagem informando destino(s), datas e preferências.

**RF02 – Definição de preferências**  
O sistema deve permitir que o usuário selecione seus interesses (ex: cultura, gastronomia, lazer, etc.) para personalização do roteiro.

**RF03 – Geração automática de roteiro**  
O sistema deve gerar automaticamente um roteiro de viagem com base nas informações fornecidas pelo usuário.

**RF04 – Estimativa de custos**  
O sistema deve apresentar estimativas de custos relacionadas à viagem, incluindo hospedagem, transporte e atrações.

**RF05 – Comparação de transporte**  
O sistema deve permitir a comparação entre diferentes meios de transporte, considerando custo e tempo de deslocamento.

---

## 🇧🇷 Requisitos Não Funcionais

**RNF01 – Usabilidade**  
O sistema deve possuir uma interface intuitiva e de fácil utilização.

**RNF02 – Desempenho**  
O sistema deve gerar o roteiro em até 5 segundos após solicitação.

**RNF03 – Disponibilidade**  
O sistema deve estar disponível 24/7.

**RNF04 – Escalabilidade**  
O sistema deve suportar múltiplos usuários simultâneos.

**RNF05 – Segurança**  
O sistema deve garantir a proteção dos dados do usuário.

---
